# Matrix-Build Output Staging

`P57.2` creates the runtime location where Patchworks matrix-build outputs will
land after the FEMIC-emitted MKRF XML is wired and the matrix build is run.

These tracks now belong to the current **MKRF PoC benchmark/intermediate**
runtime package.

Current boundary:

- generated track tables here are PoC benchmark outputs, not final canonical
  rebuild outputs; and
- archival legacy `data/legacy_mkrf/compiled_tracks/` remains comparison
  evidence only, not generated output for this directory.
